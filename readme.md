# Animated Website

This is an animated website built using HTML, CSS, JavaScript, and GSAP.

### Usage

To view the website, simply open the index.html file in a web browser.

### Technologies used

* HTML
* CSS
* JavaScript
* GSAP
* Locomotive(For Smooth Scrolling)
* Scroll Triggers
